Basics of Linux

Master the command line, understand the system, and power up your tech skills.

Start Date: 19th May  •  Instructor: Syed Hamza Ali  •  Duration: 1 Month (3 Days/Week)
Course Fee: Rs. 3000 (50% OFF for FGEI students & faculty!)

What You'll Learn

  • Navigate directories in Linux
  • Explore major Linux distributions
  • Perform basic administration tasks
  • Write simple Bash scripts
  • Manage file permissions and users

Weekly Breakdown

Week 1: Getting Started with Linux

  • Session 1: Linux History, Distributions, Live Boot
  • Session 2: Shell Basics, Terminal Usage, Common Commands
  • Session 3: Filesystem Structure, Navigation

Week 2: Files, Folders & Permissions

  • Session 4: File/Directory Commands: cp, mv, rm, mkdir
  • Session 5: Viewing & Editing Files with nano and vim
  • Session 6: Understanding chmod, chown, rwx permissions

Week 3: Users & System Tools

  • Session 7: Adding/Removing Users, Groups, Passwords
  • Session 8: Monitoring Processes with ps, top
  • Session 9: Disk Usage, Mounting, Resource Monitoring

Week 4: Tools & Scripts

  • Session 10: Package Managers (apt, yum)
  • Session 11: Networking Basics: ifconfig, ping, netstat
  • Session 12: Shell Scripting: loops, variables, automation

Final Assessment

At the end of Week 4, you'll complete a hands-on practical or quiz to test your skills. A certificate of completion will be awarded.
